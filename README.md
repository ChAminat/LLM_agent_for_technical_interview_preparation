# Проект: Ассистент по подготовке к собеседованиям


**AI Interview Agent** - система проведения технических собеседований, которая сочетает в себе Telegram-бота для взаимодействия с пользователем и LLM-агента для подбора вопросов и анализа ответов с использованием RAG. Система автоматизирует процесс технического интервью, предоставляя персонализированные вопросы и объективную оценку ответов кандидата.

## Ключевые возможности:
- Персонализированные собеседования по выбранной теме и уровню сложности
- База знаний с использованием RAG для генерации релевантных вопросов
- Автоматическая проверка ответов с детальным разбором
- Гибкая настройка параметров собеседования
- Интерактивное взаимодействие через Telegram-бот

## Компоненты системы

#### 1) Telegram Bot

| Команда | Описание |
|---------|-----------|
| `/start_interview` | Начало нового интервью (с чистого листа) |
| `/finish` | Прерывание текущего интервью |
| `/next_question` | Отправка нового вопроса пользователю |
| `/ask_question` | Пользователь задает свой вопрос |
| `/change_difficulty` | Смена уровня сложности (Junior/Middle/Senior) |
| `/change_topic` | Смена области вопросов |

При создании Telegram-бота использовался фреймворк Aiogram

#### 2) Interview Agent

**Основные функции:**
- **Прием информации** от пользователя: Имя, Тема собеседования, Уровень сложности
- **Обработка запросов** из Telegram-бота
- **Передача параметров** собеседования в RAG систему (тема, сложность)
- **Получение вопросов и эталонных ответов** из RAG
- **Возврат вердикта** в бот

#### 2) RAG Agent

**Основные функции:**

- **Поиск вопросов** из RAG соответсвующих теме и уровню сложности 
- **Поиск эталонных ответов** из RAG + из статей arxiv
- **Анализ ответа пользователя**, сравнение его с эталоном, обратная связь и корректировки
- **Анализ истории собеседования** при необходимости

Для реализации RAG использовался фреймворк llama_index. Основная LLM-модель - mistral-small-latest.
